# faceApp
**Приложение обнаруживает человека и накладывает фильтр на лицо человека, перерисовывает всю фотографию под определенный стиль, добавляет обекты (очки, украшения).**
+ Приложение/сервис основано на работе существующей нейросети, которая распознает человеческие лица, добработанной для задач редактора фотографий.
+ Для данного приложения добавлен desktop - интерфейс, написанный с помощью Python GUI библиотек.

+ Работа выполнялась в команде, в рамках проекта для III Зимней научной сессии Студенческого научного общества НИЯУ МИФИ и заняла первое место в секции "Программная инженерия", [`диплом`](./docs/CNO_programm_engineering.pdf).
